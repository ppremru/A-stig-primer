# STIG Evaluation Automation
This repository provides a unified workflow for evaluating STIG compliance across:
* **RHEL** (via OpenSCAP)
* **Windows** (via SCC/Evaluate-STIG)
* **Network** (via Configuration Pull)

## Getting Started
1. Sync this project to AAP.
2. Ensure your Execution Environment includes 'openscap-scanner'.
3. Run playbooks from the /playbooks directory.
